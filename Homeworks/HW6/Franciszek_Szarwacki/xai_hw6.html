Report¶

Task 1: Permutation-based Variable Importance for XGBoost model¶

I trained an XGBClassifier model with default parameters on yeast_me2 dataset. I then used the dalex package to perform Permutation-based Variable Importance calculation. Here are the results in terms of loss change as well as ratios.

We can clearly see features 0 and 2 as most important, while 4 and 5 are least important.

Task 2: Different models¶

I trained a bigger (deeper and more trees), a smaller (shallower and less trees) XGBClassifier and a SVM model. Then I calculated PVIs for all of them.

The results are quite interesting. The two most important features stay the same: 0 and 2, the SVM model has all of the feature importances quite similar to basic XGBoost model. The bigger and smaller XGBoost model however change the importance of some features (ex. 1). Both of those models have negative importances, which is quite interesting. In the case of the bigger model it may be a result of overfitting, however in the smaller model it is a little bit mysterious.

Task 3: other feature importance metrics for XGBoost¶

I have also used the other feature importance metrics which were implemented in xgboost package. Those are:

  • weight
  • total gain
  • total cover

I have also used the shap.explainers.Tree class to derive feature importance based on SHAP values from the TreeSHAP algorithm.

As we can see, the most important features are 0, 2, 1, 3 with the exception of weight method, where the order is a little different, however the values on positions from 2 to 5 are very similar. SHAP values have some small differences - 6 is more important than 1, although here the difference in values is also very small.

Generally speaking, all the methods agree on the unimportance of features 4 and 5.

from sklearn.metrics import accuracy_score, confusion_matrix, f1_score, roc_auc_score, recall_score, precision_score

def evaluate_method(method_name, y_test, y_eval, verbose=False):
    results = {
        func.__name__: func(y_test, y_eval) for func in [
            accuracy_score,
            confusion_matrix,
            f1_score,
            roc_auc_score,
            recall_score,
            precision_score
        ]
    }

    if verbose:
        print(method_name, ' results:')
        for func_name, score in results.items():
            print("\t", func_name, ": ", score)

    return results
