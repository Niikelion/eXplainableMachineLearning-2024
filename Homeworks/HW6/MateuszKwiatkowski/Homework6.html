Task¶

Subtask 0: Train model¶

For my experiments, I used the Magic Telescope dataset, which is used to classify whether gamma radiation has been detected, based on advanced measurements. All variables in this dataset are continuous.

On this data, I trained an XGBoost model with 100 estimators and a maximum depth of 3. Before training, I divided the set into training and test sets to then evaluate the model. I obtained an accuracy of 0.8723 and an AUC of 0.9318.

Subtask 1: Calculate PVI¶

For the trained XGBoost, I calculated the PVI for every variable and visualised it on a plot:

The coefficient for fAlpha is very high, which means that once the data in this column is permuted, the quality of the model significantly decreases. We can conclude from this that this is an important variable. I obtained the lowest coefficients for fM3Trans and fAsym, which may mean that these variables are not important in this classification problem or are strongly correlated with other variables.

Subtask 2: More models¶

For next experiments, I trained a further 3 models on the same dataset: XGBoost with 50 estimators and a maximum depth of 2, XGBoost with 250 estimators and a maximum depth of 5 and Random Forest with 100 estimators. The large XGBoost and Random Forest have very similar evaluation results: over 0.87 accuracy and over 0.93 AUC. I also calculated the PVI for these models and compared them with the first XGBoost.

The most important observation that emerges from the above comparison is that for these 4 models the variables appears in the same order of importance. The values of the individual coefficients of course differ, because we obtained different loss values for the different models.

Subtask 3: Comparison¶

We can use Gini impurity to indicate which variables are most important, as this is the value we are trying to minimise. If a tree-based model has a very high impurity it means that there are many observations from both classes in its leaves. For this loss function I did a similar analysis to the one above and looked at which variables have the greatest impact on the model.

We can see that, in this case too, the variable fAlpha proved to be the most important, while fAsym and fM3Trans are insignificant.

Finally, I prepared an analysis using the TreeSHAP algorithm:

In the graph above, we can observe some interesting facts. The variable fAlpha has a strong influence on the model, as its high values clearly decrease the model prediction and its low values increase it. The variable fSize behaves in a very similar way. In the case of fWidth, small values do not have much impact on the prediction, but its large values change the output a lot. For fAsym and f3MTrans, large values increase the prediction, but it is a very small effect.
