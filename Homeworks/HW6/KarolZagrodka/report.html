Different metric¶

In [4]:
Image("imgs/xgboost_comp_acc.png")
Out[4]:

By using different metric we can see how it affects PVI. We should always choose to analyze the metric we are trying to optimize. Here the new metric is $1 - accuracy$. We can see in the second plot that the values are smaller - so the changes in accuracy are not that big. Also, both metrics predicted free_sulfur_dioxide as the most important feature. However, volatile_acidity is in second place in the first plot and nowhere to be seen for second loss.

Three models comparison¶

I decided to compare three Random Forest Classifiers but with very different number of estimators:

  • 1
  • 50
  • 10000

Idea behind is that we want to see how the PVI looks like for under-fitted, correct and over-fitted models.

In [6]:
Image("imgs/rfcs.png")
Out[6]:

We can see that for only 1 estimator free_sulfur_dioxide is not that significant, also other features have very similar values of PVI. There is big correspondence for Random Forest Classifier with 50 and 10000 estimators. Maybe the one with 50 is already over-fitted? Or PVI doesn't change that much for over-fitted models?
